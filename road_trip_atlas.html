Date / route / notes
Prototype note: county borders include a rendering mode selector. Hybrid mode combines the normalized county-boundary mesh with a separate no-fill county-stroke safety layer so missing shared borders remain visible while the outer state outline remains separate. This build also prioritizes interstates and selected statewide corridors for quick-reference readability, with route shields forced above the other map layers.
This is a layout and data prototype. County boundaries are real embedded vectors. Use the printable size selector and layout proportion and map viewport sliders to test portrait and wide versions of 9:16, half-letter, letter, A5, and 4 x 6 formats, plus vertical and horizontal map/checklist layouts. The map is intended as a broad quick-reference print piece, not a turn-by-turn GIS overlay. Interstate lines remain the primary navigation layer, while the U.S.-route layer is limited to selected statewide corridors that help orient the viewer. Route shields are rendered above all other map layers for readability, are slightly smaller than before, and use stronger collision-avoidance logic so they stay clearer around cities and around one another. The capital is shown with a star, and the city layer uses curated opinionated orientation-city sets for Florida, Georgia, Alabama, and Pennsylvania. County-boundary detection has also been expanded so a small number of straight shared borders that align with state-border direction are more reliably drawn as county lines instead of appearing like heavier outline continuations.
